Capitalization
--------------

Capitalization of some keywords, such as::

  MODULE ...
    USE xxx
    ...
  CONTAINS
  
  SUBROUTINE sub
  END SUBROUTINE sub
  
  FUNCTION fun
  END FUNCTION fun

serves the purpose of allowing editor search patterns such as ``'USE xxx'``, 
``'E sub'``, or ``'N fun'``.
