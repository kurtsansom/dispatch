.. _python:

Python support
==============

To import Python modules, add the full path of the DISPATCH directory ``utilities/python/``
to the Python path (see :ref:`environment`), and do::

   import dispatch
   import dispatch.select
   import dispatch.graphics
   import dispatch.yt
