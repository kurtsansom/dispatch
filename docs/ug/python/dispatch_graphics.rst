``dispatch.graphics`` module
-----------------------------------

``dispatch.graphics`` contains support functions for visualizing
snapshot data.

Documentation::

  help(dispatch.graphics)
  help(dispatch.graphics.imshow)

Note for example ``dispatch.graphics.imshow(im)``, which is similar
to ``matplotlib.pyplot.imshow(im)``, but uses Fortan index conventions.
