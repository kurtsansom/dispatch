Windows: Cygwin
---------------

In Windows, a simple way to get access to gfortan and other tools that are needed (git, make, ..)
is to install Cygwin (cygwin.org).  Choose the 64-bit version, and be sure to include::

  gcc-core
  gcc-fortran
  gcc-debuginfo
  openmpi
  openmpi-debuginfo
  libopenmpi-devel
  libopenmpi40
  libopenmpifh40
  libopenmpiuse08_40
  libopenmpiusetkr40
  make
  git

and maybe more (shells, SSH, ...)
